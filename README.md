A modern, responsive, and beautiful financial dashboard built with **Next.js 14** (App Router) and **Tailwind CSS**. It provides a clear overview of incomes, expenses, and current balance.

## Features
- Server-Side Rendering (SSR) via Next.js App Router
- Beautiful UI built with Tailwind CSS (Fully Responsive)
- Lucide React for crisp, modern icons
- Dark Mode support out-of-the-box

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/czuwnio/expense-tracker-dashboard.git
cd expense-tracker-dashboard
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
